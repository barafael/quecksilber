# Reference Images

## Mercury Spacecraft Panels
- `01_mercury_main_instrument_panel_diagram.png` — Technical diagram of the main instrument panel: round analog gauges (attitude, altitude, rate), periscope display, rows of round status indicators, toggle switches, hand controller
- `02_mercury_console_panels_diagram.png` — Left and right console panels: rows of labeled indicator lights, rotary selector knob
- `03_mercury_cabin_interior_photo.jpg` — Interior view of crew station from above, showing panel arrangement surrounding pilot
- `04_mercury_interior_arrangement.png` — Spacecraft interior arrangement diagram
- `13_mercury_panel_closeup_cooper.jpg` — Cooper examining spacecraft #20 instrument panel, visible round gauges and circular indicator rows
- `15_mercury_hand_controller.jpg` — Three-axis hand controller diagram
- `16_mercury_panel_print_osu.jpg` — Color print of Mercury control panel (Glenn Archives)
- `17_mercury_navigational_aids.png` — Navigational aids diagram

## Mission Control Consoles
- `05_mission_control_room.jpg` — Williams and Kraft in Mercury mission control (B&W)
- `06_mission_control_capcom_console.jpg` — Shepard at CAPCOM console with world tracking map behind
- `07_mercury_control_console_museum_1.jpg` — Museum restoration: full room with world map display and consoles
- `08_mercury_control_console_museum_2.jpg` — Wide view: world tracking map with orbital paths, consoles below
- `09_mercury_control_console_museum_3.jpg` — Close-up: world map with labeled clocks (GMT, Orbit Number, Countdown, Elapsed Time), console indicator grids
- `10_mercury_control_console_museum_4.jpg` — Close-up of two consoles: left has rows of backlit indicator buttons + CRT; right has colored indicator light grid (red/green/amber)
- `11_mercury_control_console_museum_5.jpg` — Console detail: small CRT, rows of backlit buttons, clocks in background
- `12_mercury_control_console_museum_6.jpg` — Multiple consoles with rotary phone, working environment context
- `14_mercury_interior_layout.jpg` — Mercury cabin interior layout
- `18_mission_control_console_bullock.jpg` — Bullock Museum: full mission control console with backlit button rows, CRT screens, toggle switches, rotary knobs
